# SolLock — Solana NFC Access Control System

Watch me build it: https://youtu.be/Nb1tHa4NIRw

## Overview
SolLock links Solana wallet identity to physical access hardware.  
Using an ESP32 + PN532 NFC module, SolLock can detect a wallet address stored in an NFC tag or a phone’s NFC app, send it to a backend, verify authorization, and unlock a relay-driven lock.
